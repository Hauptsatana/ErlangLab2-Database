
Лабораторная №2

Реализовать функции работы с БД. БД - список из кортежей {Key, Value} 
Все функции кроме get возвращают новый список.
Функции:
* new() - пустая БД
* put(Key, Value, Db)
* delete(Key, Db)
* get(Key, Db)
* find(Value, Db) - вернуть список всех ключей по указанному значению

Пример:
Db = db:new().
Db1 = put(1, "Vasya", Db).
Db2 = put(2, "Kolya", Db1).
Db3 = delete(3, Db2).

Кроме того, в общем виде реализовать функции 
* filter(Pred, List) - возвращает новый список из элементов List, удовлетворяющих Pred
* reduce(Func, List) - применяет указанную функцию с накоплением ко всем элементам списка. Для списка [a, b, c, d] результат - f( f( f(a,b), c), d).
